#### Kodev

Kodev is a programming learning platform for Turkish people. I created it due to the absence of a proper and static learning platform in the Turkish language. It is a currently ongoing project. It is being created by me and my friend, [Furkan](https://github.com/kontgerilla), and we are doing our best to keep it going.

#### SSF

SSF is a simple piece of code that allows you to execute JavaScript code that helps you add CSS. It contains a prototype function, and a global function which takes a CSS selector as input.

#### WIS

WIS is a private project made by me. It is an alternative to Selenium, as it uses websockets as a way to communicate between the browser and the server.
